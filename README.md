## ‚öíÔ∏è Develop

> Requirement:
>
> **Node >=16**
>
> **[NPSSO](https://psn-api.achievements.app/authentication/authenticating-manually)**

```shell
# copy a .env.local file
cp .env.example .env.local
```

Add your NPSSO on .env.local file
```shell
NPSSO=YOUR_NPSSO_KEY
```

```shell
# install dependencies
pnpm install
# or
yarn install
# or
npm install

# start the service
npm run dev
```
